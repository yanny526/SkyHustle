Bot Commands

/start    - Start the bot and show welcome message
/help     - Show available commands
/status   - Show player status and resources
/build    - Build structures
/train    - Train units
/research - Research technologies

Telegram Bot Instructions

1. Find the bot on Telegram with the username you specified when creating it

2. Start a conversation with the bot by clicking "Start" or sending /start

3. Use the commands listed above to interact with the bot

Technical Information

Bot is using:

  • python-telegram-bot for Telegram integration
  • gspread for Google Sheets integration
  • Flask for this web dashboard
